## Controls

Left/Right: `LEFT ARROW` and `RIGHT ARROW`

Forward/Back: `UP` and `DOWN`

Ascend: `SPACE`

Descend: `LEFT SHIFT`


Rotations: keys `1`-`6`


Zoom in: `+`

Zoom out: `-`
